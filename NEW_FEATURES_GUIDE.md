### Feature 2: Multi-Project Management

#### What It Does:
Manage multiple construction sites/projects independently within the same application.

#### Key Benefits:
- ‚úÖ Create unlimited construction projects
- ‚úÖ Each project has its own financial data
- ‚úÖ Switch between projects easily
- ‚úÖ No data mixing between projects
- ‚úÖ Track multiple sites simultaneously

#### How to Use:

##### Creating Your First Project:
1. **Login** to the application
2. You'll land on the **Projects** page
3. Click **"New Project"** button
4. Fill in the form:
   - **Project Name:** e.g., "Residential Building - Phase 1"
   - **Total Committed Amount:** e.g., 5000000 (total contract value)
   - **Description:** Optional project details
5. Click **"Create Project"**
6. Your project is created and automatically selected!

##### Managing Multiple Projects:
1. Go to **Projects** page (first item in sidebar)
2. See all your projects in a grid layout
3. Each card shows:
   - Project name and status
   - Total contract amount
   - Payment progress (percentage)
   - Received vs remaining amounts
   - Total expenses
   - Net balance
4. Click any project card to **switch to that project**
5. Active project is highlighted with a blue border

##### Editing a Project:
1. Find the project card
2. Click the **Edit icon** (pencil)
3. Update details:
   - Project name
   - Total committed amount
   - Description
   - Status (Active/On-Hold/Completed)
4. Click **"Update Project"**

##### Deleting a Project:
1. Find the project card
2. Click the **Delete icon** (trash)
3. Confirm deletion
4. Project and all its data will be removed

##### Project Status:
- **Active:** Currently ongoing project (green)
- **On-Hold:** Temporarily paused (orange)
- **Completed:** Finished project (blue)

### Feature 3: Total Committed Amount Tracking

#### What It Does:
Track the total contract amount and monitor payment progress against it.

#### Key Benefits:
- ‚úÖ Know total contract value
- ‚úÖ See how much has been paid
- ‚úÖ Track remaining balance to collect
- ‚úÖ Visual progress indicators
- ‚úÖ Percentage completion at a glance

#### How to Use:

##### Setting Total Committed Amount:
1. When creating a project, enter the **Total Committed Amount**
2. This is the total contract value agreed with the client
3. Example: If the total construction cost is ‚Çπ50,00,000, enter 5000000

##### Viewing Payment Progress:

**On Projects Page:**
- Each project card shows a progress bar
- See percentage of payment received
- View total committed, received, and remaining amounts

**On Dashboard:**
- Large **Payment Progress Card** at the top
- Shows:
  - Percentage received (e.g., 65.5%)
  - Total contract amount
  - Amount received so far
  - Remaining balance to collect
- Visual progress bar fills based on percentage

##### Understanding the Metrics:
- **Total Contract:** The full contract amount
- **Received:** Sum of all payments in
- **Remaining:** Total Contract - Received
- **Percentage:** (Received / Total Contract) √ó 100

##### Example Scenario:
