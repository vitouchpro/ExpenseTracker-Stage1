## ğŸ¯ **Quick Start**

### **Step 1: Open the Form**
- Go to **Payments In** or **Payments Out** page
- Click **"Add Payment"** or **"Add Expense"** button

### **Step 2: Fill in Details**
- Enter amount, date, description, etc.
- Scroll down to the **"Attachments"** section

### **Step 3: Upload Files**
- Click **"Upload Files"** button
- Select one or more files from your computer
- Supported formats: Images, PDF, Word, Excel, Text
- Maximum 5 files per transaction
- Maximum 5MB per file

### **Step 4: Review & Submit**
- See your uploaded files in the list
- Preview or remove files if needed
- Click **"Add Payment"** or **"Add Expense"** to save

## ğŸ’¡ **Tips & Best Practices**

### **For Best Results:**
1. **Use PDF for documents** - Better compatibility and smaller file size
2. **Compress images** - Keep files under 2MB when possible
3. **Use clear filenames** - e.g., "Invoice_ABC_Corp_Jan2025.pdf"
4. **Attach all relevant documents** - Receipts, invoices, contracts
5. **Check file size** - Must be under 5MB

### **What to Attach:**

**Payments In:**
- Client invoices
- Payment receipts
- Bank transfer confirmations
- Contracts

**Payments Out:**
- Supplier invoices
- Purchase receipts
- Bills
- Delivery notes

## âš ï¸ **Common Errors**

### **"File size must be less than 5MB"**
- **Solution:** Compress the file or use a smaller version
- **Tip:** Use online tools to compress PDFs/images

### **"File type not supported"**
- **Solution:** Convert to a supported format (PDF, JPG, PNG, etc.)
- **Tip:** Use PDF for documents, JPG for images

### **"Maximum 5 files allowed"**
- **Solution:** Remove some files or combine documents
- **Tip:** Merge multiple PDFs into one file

## ğŸ’¾ **Backup & Restore**

### **Backups Include Attachments:**
- All attached files are included in backups
- No need to save files separately
- Restore brings back all attachments

### **To Backup:**
1. Go to **Settings** page
2. Click **"Backup Data"**
3. Save the .ttf or .json.gz file
4. All attachments are included

### **To Restore:**
1. Go to **Settings** page
2. Click **"Restore Data"**
3. Select your backup file
4. All attachments are restored

## â“ **FAQ**

### **Q: How many files can I attach?**
A: Up to 5 files per payment/expense.

### **Q: What's the maximum file size?**
A: 5MB per file.

### **Q: Can I attach photos from my phone?**
A: Yes! Use your phone's camera or gallery.

### **Q: Are attachments included in backups?**
A: Yes, all attachments are automatically included.

### **Q: Can I remove an attachment later?**
A: Yes, edit the record and remove the file.

### **Q: What happens if I upload a large file?**
A: You'll see an error. Compress the file and try again.

### **Q: Can I preview Word/Excel files?**
A: No, but you can download them to view.

### **Q: Are my files secure?**
A: Yes, files are stored locally in your browser.
